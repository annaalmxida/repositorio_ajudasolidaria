ATIVIDADE PRÁTICA – DESENVOLVIMENTO MOBILE
Curso Técnico em Desenvolvimento de Sistemas
Framework: Flutter | Linguagem: Dart
PROJETO: SOS Clima — Rede Solidária
Eventos climáticos extremos têm provocado alagamentos, quedas de árvores, interrupção no
fornecimento de água e energia elétrica, além do aumento de doenças associadas à água contaminada.
Sua equipe deverá desenvolver um aplicativo móvel solidário que conecte pessoas em situação de
vulnerabilidade com voluntários.
Objetivos de Aprendizagem
• Compreender a estrutura básica de um projeto Flutter
• Criar interfaces com widgets do Material Design
• Implementar navegação entre telas
• Utilizar StatefulWidget e setState
• Criar formulários com validação
• Exibir dados com ListView
• Utilizar SnackBar para feedback
• Persistir dados simples com Shared Preferences
Requisitos Funcionais
• Tela de Termos com aceite salvo em Shared Preferences
• Tela Home com navegação para todas as funcionalidades
• Formulário 'Pedir Ajuda' com validação obrigatória
• Formulário 'Quero Ajudar' com cadastro de voluntário
• Painel de Ocorrências com ListView
• Tela de Detalhe com alteração de status usando setState
